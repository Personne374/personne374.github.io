Convertions

Primitives de base

  • car ( entier ) --> caractère : retourne le caractère correspond à la valeur entière de la table ASCII
  • ord ( caractère ) --> entier : retourne la valeur ASCII d'un caractère
  • enChaine ( réel ) --> chaine : convertie un réel en Chaine
  • enChaine (entier) --> chaine : convertie un entier en Chaine
  • enEntier ( chaine ) --> entier : convertie une Chaine en entier
  • enRéel ( chaine ) --> réel : convertie une Chaine en réel
  • plancher ( réel ) --> entier : retourne la valeur plancher d'un réel
  • plafond ( réel ) --> entier : retourne la valeur plafond d'un réel
  • arrondi ( réel ) --> entier : retourne la valeur arrondie d'un réel

Autres primitives

  • aujourdhui() --> chaine : retourne sous forme de chaine la date du jour au format jj/mm/aaaa
  • jour(chaine) --> entier : retourne la partie jour d'une chaine (format jj/mm/aaaa)
  • mois(chaine) --> entier : retourne la partie mois d'une chaine (format jj/mm/aaaa)
  • annee(chaine) --> entier : retourne la partie jour d'une chaine (format jj/mm/aaaa)
  • estRéel (chaine ) --> booléen : indique si la chaine peut être convertie en réel
  • estEntier (chaine ) --> booléen : indique si la chaine peut être convertie en entier
  • hasard (entier) --> entier : retourne une valeur entière prise au hasard sur l'intervalle [0; n[

Conventions d'écriture

Casse

  • Nom d'un Algorithme : UpperCamelCase
  • Nom d'une CONSTANTE : UPPER_CASE
  • Nom d'une variable : lowerCamelCase
  • Mot-clé du langage : lowercase

Police de caractère

  • police proportionnelle : la largeur des caractères dépend du caractère
  • police non-proportionnelle : tous les caractère ont la même largeur

Indentation

  • Style GNU
  • Style Whitesmiths
  • Style Allman
  • Style K&R

//style K&R
if (condition){
	instruction
}



//style Allmand
if (condition)
{
	instruction
}



//style GNU
if (condition)
	{
		instruction
	}

  • Paramètre formel : variable utilisée dans le sous-programme (durée d'exécution)
  • Paramètre effectif : variable fournies lors de l'appel du programme (recopié dans un paramètre formel ou en sortie dans une variable)

  • s : sortie
