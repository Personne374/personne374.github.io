Table de caractères

Caractères d'échappement

  • \a : cloche d'alarme
  • \b : arrière-espace
  • \t : tabulation
  • \r : retour chariot
  • \v : tabulation verticale
  • \f : saut de page
  • \n : nouvelle ligne
  • \e : caractère d'échappement
  • \NNN : caractère ascii (où NNN est un caractère octal)
  • \xNN : caractère ascii (où NN est un caractère hexadécimal)
  • \cX : X est la lettre du caractère de controle (ctrl-C)

Caractères algo

  • ← : affectation
  • × : multiplication
  • © : concaténation
  • ≠ : différent de
  • ≥ : plus grand que
  • ≤ : plus petit que
