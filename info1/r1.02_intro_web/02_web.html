Abcdaire

HTML

ABC

  • a : liens, <a href="lien" target="_blank"> </a>
  • area : zones définies pour la balise map, <area shape="square" coords="1,2,3,4"/>
  • body : corp de page, <body></body>
  • b : gras, <b> </b>
  • big : aggrandir, <big> </big>
  • blockquote : citation, <blockquote> </blockquote>
  • br : retour à la ligne, <br>
  • <!-- commentaire --> : commentaire
  • caption : titre de tableau, <caption> </caption>
  • colgroup : groupement de colonnes, <colgroup> </colgroup>
  • col : colonne d'un groupement de colonnes,<col> </col>

DEF

  • dl : liste de definition, <dl> </dl>
  • dt : nom d'une definition, <dt> </dt>
  • dd : definition du terme, <dd> </dd>
  • div : mettre en forme un bloc, <div class=""> </div>
  • em : relief, <em> </em>
  • figure : groupe d'image, <figure> </figure>
  • figcaption : description de figure, <figcaption> </figcaption<>

GHI

  • h1 : titre, <h1> </h1>
  • h2 : titre, <h2> </h2>
  • h3 : titre, <h3> </h3>
  • h4 : titre, <h4> </h4>
  • h5 : titre, <h5> </h5>
  • h6 : titre, <h6> </h6>
  • head : entête, <head> </head>
  • hr : separateur, <hr size="epaisseur" width="longueur" align="alignement">
  • html : page web, <html> </html>
  • img : image, <img src="lien.ext" alt="pas-d-image" title="c-une-image">
  • i : italique, <i> </i>

JKL

  • li : élément d'une liste, <li> </li>
  • link : lien entete, <link> </link>
  • link icon : logo page <link rel="icon" href="image.ico">
  • link stylesheet : lien css, <link rel="stylesheet" href="style.css" media="all" type="text/css">

MNO

  • map : zone cliquable sur une image, <map name=""> </map >
  • mark : surligner, <mark> </mark>
  • meta : info dans l'entête, <meta> </meta>
  • meta charset : encodage, <meta charset="utf-8">
  • meta author : info auteur, <meta name="author" content="Maxime Lemoine">
  • meta http-equiv : reactualiser la page, <meta http-equiv="refresh" content="X">
  • ol : liste ordonnée, <ol type="A|a|I|i|1" start="3"> </ol>

PQR

  • p : paragraphe, <p> </p>
  • pre : respecter format, <pre> </pre>

STU

  • s : barrer, <s> </s>
  • span : mettre en forme un morceau, <span class=""> </span>
  • small : retrecir, <small> </small>
  • sup : exposant, <sup> </sup>
  • sub : indice, <sub> </sub>
  • tt : machine à écrire, <tt> </tt>
  • title : titre page, <title> </title>
  • table : tableau, <table> </table>
  • tr : ligne de tableau, <tr rowspan="longueur"> </tr>
  • th : cellule d'en-tête, <th> </th>
  • td : cellule, <td colspan="longueur"> </td>
  • thead : section d'en-tête du tableau, <thead> </thead>
  • tbody : section du corps du tableau, <tbody> </tbody>
  • tfoot : section du pied du tableau, <tfoot> </tfoot>
  • u : souligner, <u> </u>
  • ul : liste non ordonnée, <ul> </ul>

YZ

CSS

ABC

  • align : allignement ("left" "right" "center")
  • alink : couleur des liens activés
  • background-color : couleur de fond
  • background-image : image du fond ("url('image.img')")
  • background-position : positionnement
  • background-repeat : image à répeter ("no-repeat")
  • bottommargin : largeur de la marge du bas
  • border : carractéristiques des bordures (taille, forme, couleur)
  • border-width : largeur de la bordure (thin, medium, thick)
  • border-color : couleur de la bordure
  • border-style : style de la bordure (dotted, dashed, solid, double, groove, ridge, inset, outset)
  • border-radius : arrondis des bordures
  • box-shadow : ombre d'un bloc avec 4 valeurs (+1 facultatif) dans l'ordre : décalage H, décalage V, adoucissment, couleur (, "inset")
  • bgcolor : couleur du fond
  • bgproperties : fixe l'image de fond (avec fixed)
  • color : couleur du texte

DEF

  • display : type d'affichage ("display: table-cell")
  • font : regroupe toutes les modification sur la police ("font : police taille type")
  • font-style : type (bold, italic....)
  • font-family : police d'écriture ("font-family : arial, verdana, sans-serif")
  • font-size : taille de l'écriture
  • font-variant : variantes d'une police (small-caps, normal)
  • font-weight : type d'écriture (bold, italic)
  • float : positionnement (left, right, top, bottom)

GHI

  • height : hauteur

JKL

  • leftmargin : largeur de la marge de gauche
  • letter-spacing : espacement entre les cararctères
  • link : couleur des liens
  • line-height : longueur des lignes

MNO

  • margin : largeur de la marge autour

PQR

  • rightmargin : largeur de la marge de droite

STU

  • size : taille
  • start (pour < ol >) : valeur de départ
  • text-align : aligner le texte (left right center justify)
  • text-decoration : décoration du texte (underline, line-through, overline, none)
  • text-indent : indentation d'un paragraphe
  • text-transform : convertion du texte (uppercase, capitalize, lowercase, none)
  • text-shadow : ombre d'un text avec 4 valeurs (+1 facultatif) dans l'ordre : décalage H, décalage V, adoucissment, couleur (, "inset")
  • type (pour < ul >) : mise en forme ("disc" "circle" "square")
  • type (pour < ol >) : mise en forme ("A" "a" "I" "i" "1")
  • scroll : définit s'il y a des barres de défilement ("yes" ou "no")
  • topmargin : largeur de la marge du haut
  • text : couleur du texte
  • text-align : alignement du texte (center, left, right)

VWX

  • value (pour < li >) : valeur de la balise
  • vlink : couleur des liens déjà visités
  • width : longueur

YZ
